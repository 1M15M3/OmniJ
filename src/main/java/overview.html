The bitcoin-spock project was created for writing Spock tests of Bitcoin RPC and Omni Core RPC (formerly Master Core) functions. In addition to the Spock specifications there is a collection of test support classes written in Java and Groovy. We are in the process of breaking out these support classes into their own subprojects and JAR files, as follows:

bitcoin-rpc:
  • A client library for talking to bitcoind via JSON-RPC
  • A library for writing command-line tools that talk to bitcoind via JSON-RPC
omnij-rpc:
  • Omni Protocol extensions to the JSON-RPC client for Omni Core
  • Omni Protocol-specific command-line tools
  • Classes for fetching, parsing, and comparing Omni Protocol consensus information
omnij-core:
  • Classes defining abstractions from the Omni Protocol Specification
