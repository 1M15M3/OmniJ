The bitcoin-spock project was created for writing Spock tests of Bitcoin RPC and Mastercoin RPC functions. In addition to the Spock specifications there is a collection of test support classes written in Java and Groovy. Some of these support packages should probably be broken out in to separate projects, modules, and/or jars. For the time being it is simplest to keep them all in one project/module/jar. The functionality consists of:
  • A client library for talking to bitcoind via JSON-RPC
  • A library for writing command-line tools that talk to bitcoind via JSON-RPC
  • Mastercoin extensions to the JSON-RPC client for Master Core
  • Classes defining abstractions from the Master Protocol Specification
  • Mastercoin-specific command-line tools
  • Classes for fetching, parsing, and comparing Master Protocol consensus information
