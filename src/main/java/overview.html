A Java/JVM implementation of the http://www.omnilayer.org[Omni Layer], an open-source, fully decentralized asset creation platform built on the Bitcoin blockchain.

The OmniJ project was initially created for writing Spock tests of Bitcoin RPC and Omni Core RPC (formerly Master Core) functions.

In addition to the Spock specifications there is a collection of test support classes written in Java and Groovy. OmniJ currently consists of three subprojects/JARs, each of which includes unit and integration tests:

bitcoin-rpc:
  • A client library for talking to bitcoind via JSON-RPC
  • A library for writing command-line tools that talk to bitcoind via JSON-RPC
  • Some sample command-line tools
  • Integration tests (Spock Specs) and test support classes for testing Bitcoin Core RPCs
omnij-rpc:
  • Omni Protocol extensions to the JSON-RPC client for Omni Core
  • Omni Protocol-specific command-line tools
  • Classes for fetching, parsing, and comparing Omni Protocol consensus information
  • Integration tests (Spock Specs) and test support classes for testing Omni Core RPCs
omnij-core:
  • Classes defining abstractions from the Omni Protocol Specification
  • OmniTxBuilder and supporting classes to create and sign Omni Protocol transactions
